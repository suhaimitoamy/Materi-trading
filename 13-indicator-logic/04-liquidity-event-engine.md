# Liquidity Event Engine

## Tujuan
Engine ini dipakai untuk mendeteksi event seperti:
- sweep
- grab
- reclaim
- level likuiditas yang tersentuh

---

## Fungsi utama
- memetakan buy-side dan sell-side liquidity
- menandai saat level penting disentuh atau ditembus
- membantu sistem membedakan event awal dan shift lanjutan

---

## Ringkasan
Liquidity event engine membantu indikator membaca momen pengambilan likuiditas dengan lebih terstruktur.
