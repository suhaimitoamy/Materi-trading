# Premium

## Apa itu
Premium adalah area bagian atas dari dealing range.

## Kenapa penting
Area ini sering lebih logis untuk mencari sell jika context mendukung.

## Cara baca sederhana
Kalau harga sudah relatif mahal terhadap range aktif, trader perlu lebih hati-hati untuk buy dan lebih tertarik mengamati sell setup yang didukung context.

## Kapan valid
Jika range yang dipakai relevan dan bias mendukung pembacaan tersebut.

## Kapan menyesatkan
Premium bukan berarti harga pasti turun. Ini hanya alat lokasi, bukan jaminan arah.

## Hubungan dengan konsep lain
- discount
- equilibrium
- dealing range
- location based entry
