# Discount

## Apa itu
Discount adalah area bagian bawah dari dealing range.

## Kenapa penting
Area ini sering lebih logis untuk mencari buy jika context mendukung.

## Cara baca sederhana
Kalau harga sudah relatif murah terhadap range aktif, trader perlu lebih hati-hati untuk sell dan lebih tertarik mengamati buy setup yang didukung context.

## Kapan valid
Jika range yang dipakai relevan dan bias mendukung pembacaan tersebut.

## Kapan menyesatkan
Discount bukan berarti harga pasti naik. Ini hanya alat lokasi, bukan jaminan arah.
