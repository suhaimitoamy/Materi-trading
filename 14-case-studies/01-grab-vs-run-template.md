# Case Study Template — Grab vs Run

## Tujuan
Template ini dipakai untuk membedakan apakah sebuah event likuiditas hanya grab atau sudah berkembang menjadi run.

---

## Checklist observasi
- liquidity mana yang disapu
- session apa saat event terjadi
- ada reclaim atau tidak
- swing terdekat pecah atau tidak
- displacement cukup atau tidak
- market retrace lalu lanjut atau kembali noise

---

## Kesimpulan akhir
Tuliskan:
- ini grab atau run
- alasan utama
- apa yang membuat pembacaan itu valid atau lemah

---

## Catatan
Template seperti ini berguna agar proses review chart tidak sekadar melihat hasil akhir, tetapi juga melihat urutan logika market.
