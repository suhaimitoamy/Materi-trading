# Liquidity Run

## Apa itu
Liquidity run adalah kondisi ketika setelah liquidity event, market benar-benar mulai delivery ke arah berikutnya.

## Cara baca sederhana
- ada sapuan likuiditas
- swing terdekat yang relevan dipatahkan
- displacement cukup kuat
- market mulai continuation

## Kenapa penting
Run punya bobot lebih tinggi daripada grab karena sudah menunjukkan kelanjutan yang lebih matang.

## Kapan valid
Lebih layak disebut run jika:
- ada event likuiditas sebelumnya
- ada break level yang relevan
- market tidak hanya bereaksi, tetapi bergerak lanjut

## Kapan menyesatkan
Kalau belum ada shift valid dan belum ada continuation, lebih aman menyebutnya grab, bukan run.

## Hubungan dengan konsep lain
- liquidity sweep
- liquidity grab
- MSS
- continuation leg
