# Level 1 — The Foundations

## Tujuan Utama

Level ini membangun pondasi belajar agar pemula tidak langsung masuk ke konsep yang berat.

Di level ini pembaca belajar cara berpikir, mengenali emosi, memahami probabilitas, membaca chart paling dasar, mengenal istilah inti, dan memahami risiko sebelum mencari entry.

## Syarat Kelulusan Level

Sebelum lanjut ke Level 2, pembaca harus bisa:

- menjelaskan kenapa tidak boleh langsung mengejar entry;
- mengenali FOMO, revenge, dan overtrade;
- memahami bahwa satu hasil tidak cukup untuk menilai proses;
- membaca candle, high, low, swing, trend, dan range sederhana;
- memahami fungsi risk, target, dan batas salah;
- membuka chart dan membuat catatan latihan sederhana.
