# Bab 06 — Run menuju Liquidity Berikutnya

## Tujuan Bab

Bab ini menjelaskan run sebagai pergerakan harga menuju liquidity berikutnya setelah liquidity sebelumnya diambil.

Setelah memahami grab dan displacement, pembaca perlu melihat bahwa market sering berpindah dari satu area liquidity ke area liquidity lain.

---

## Apa Itu Run?

Run adalah pergerakan harga setelah liquidity diambil.

Run bisa bergerak menuju:

- BSL berikutnya;
- SSL berikutnya;
- equal high;
- equal low;
- old high;
- old low;
- IRL atau ERL;
- liquidity objective yang lebih besar.

Run membantu pembaca memahami lanjutan dari proses sweep dan grab.

---

## Run Tidak Selalu Jauh

Kadang run hanya pendek menuju internal liquidity.

Kadang run bisa panjang menuju external liquidity.

Jarak run tergantung pada:

- struktur;
- DOL;
- liquidity yang belum diambil;
- kekuatan displacement;
- kondisi market sedang trend atau range;
- waktu/session.

---

## Run dalam Kondisi Bullish

Contoh bullish:

1. harga mengambil SSL;
2. muncul rejection dan displacement naik;
3. harga mulai run ke BSL;
4. target bisa berupa old high atau equal high.

---

## Run dalam Kondisi Bearish

Contoh bearish:

1. harga mengambil BSL;
2. muncul rejection dan displacement turun;
3. harga mulai run ke SSL;
4. target bisa berupa old low atau equal low.

---

## Jangan Mengejar Run yang Sudah Jauh

Pemula sering melihat run lalu takut ketinggalan.

Masalahnya, setelah run terlalu jauh, risk sering menjadi buruk.

Folder ini belum membahas entry. Namun prinsipnya tetap penting:

> Membaca run berbeda dengan mengejar run.

---

## Kesalahan Umum Pemula

- Mengejar harga setelah run sudah jauh.
- Tidak tahu target liquidity berikutnya.
- Mengira semua run akan panjang.
- Tidak membedakan run pendek dan delivery besar.
- Tidak melihat apakah run sudah mendekati objective.

---

## Latihan Praktik

Cari 10 contoh run setelah liquidity grab.

Untuk setiap contoh, tulis:

1. liquidity awal yang diambil;
2. arah run setelahnya;
3. target liquidity berikutnya;
4. apakah run pendek atau panjang;
5. apakah harga terlalu jauh untuk dikejar.

---

## Ringkasan Bab

Run adalah pergerakan harga setelah liquidity diambil. Run bisa menuju liquidity internal atau external. Pembaca harus memahami target run tanpa mengejar harga yang sudah bergerak terlalu jauh.

---

## Checklist Sebelum Lanjut

Sebelum lanjut, pembaca harus bisa:

- menjelaskan run;
- menentukan target liquidity setelah grab;
- membedakan run pendek dan run panjang;
- memahami bahwa run bukan alasan mengejar harga;
- menghubungkan run dengan DOL dari folder 02.
