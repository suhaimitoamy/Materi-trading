# Fair Value Gap

## Apa itu
Fair Value Gap adalah area ketidakseimbangan harga yang sering tertinggal setelah dorongan cepat.

## Kenapa penting
FVG sering dipakai sebagai area repricing atau area retrace yang menarik.

## Cara baca sederhana
Nilai FVG naik jika:
- lahir setelah displacement jelas
- lahir setelah event penting
- berada di lokasi yang logis
- belum terlalu sering disentuh

## Kapan valid
Kalau context besar mendukung dan area itu bukan hanya noise kecil.

## Kapan menyesatkan
Kalau setiap celah kecil dianggap FVG penting, chart akan kehilangan fokus.

## Hubungan dengan konsep lain
- imbalance
- consequent encroachment
- order block
- POI selection
