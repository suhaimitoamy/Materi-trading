# Order Block

## Apa itu
Order Block adalah area asal dorongan penting sebelum displacement yang berarti.

## Kenapa penting
OB sering dipakai sebagai area retrace atau area observasi untuk entry.

## Cara baca sederhana
Nilai OB naik jika:
- lahir setelah event penting
- lahir sebelum displacement jelas
- berada di lokasi yang logis
- masih fresh

## Kapan valid
Kalau context, lokasi, dan reaksi market mendukung.

## Kapan menyesatkan
Kalau semua candle terakhir dianggap OB, chart akan penuh area lemah.

## Hubungan dengan konsep lain
- fair value gap
- breaker block
- mitigation block
- POI selection
