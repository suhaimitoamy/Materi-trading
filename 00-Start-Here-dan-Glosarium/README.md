# 00 — Start Here dan Glosarium

Folder ini adalah pintu masuk utama untuk seluruh repo.

Fungsi folder ini bukan untuk mencari setup, tetapi untuk menyiapkan cara belajar agar pembaca tidak lompat ke materi yang terlalu berat.

---

## Tujuan Folder

Setelah menyelesaikan folder ini, pembaca diharapkan mampu:

- memahami cara memakai repo;
- mengetahui urutan belajar folder 00 sampai 10;
- membedakan istilah inti dan istilah lanjutan;
- menyiapkan chart untuk latihan;
- membaca materi tanpa merasa harus memahami semua istilah sekaligus.

---

## Cara Membaca Folder Ini

Urutan yang disarankan:

1. **Bab 01 — Mulai dari Sini**  
   Cara memakai repo dan menghindari belajar secara acak.

2. **Bab 02 — Glosarium Istilah Dasar**  
   Kamus kerja. Jangan dihafal semua sekaligus.

3. **Bab 03 — Cara Menggunakan Repo Ini**  
   Cara membaca materi, membuka chart, dan membuat catatan.

4. **Bab 06 — Panduan TradingView**  
   Setup alat belajar agar chart rapi dan mudah dilatih.

5. **Bab 07 — Panduan Belajar dan Alur Repo**  
   Peta lengkap sebelum lanjut ke folder 01.

---

## Prinsip Penting

Glosarium bukan ujian hafalan.

Jika menemukan istilah sulit, catat dulu. Istilah lanjutan akan lebih mudah dipahami setelah pembaca menyelesaikan fondasi, struktur, likuiditas, dan eksekusi dasar.

---

## Syarat Lanjut ke Folder 01

Sebelum lanjut, pembaca harus bisa:

- menjelaskan fungsi folder 00 sampai 10 secara garis besar;
- memahami bahwa folder 07 dan 09 bukan titik awal belajar;
- membuka chart dan menyiapkan workspace sederhana;
- menggunakan glosarium sebagai referensi, bukan hafalan.
