# Scoring System

## Kenapa scoring system berguna
Dalam indikator atau sistem mekanis, kadang market tidak sepenuhnya hitam putih.

Scoring system membantu menilai kualitas setup berdasarkan beberapa faktor sekaligus.

---

## Contoh faktor penilaian
- bias HTF
- struktur
- liquidity event
- MSS/BMS
- lokasi premium/discount
- kualitas POI
- session
- trigger
- chase risk

---

## Manfaat
Scoring system membantu:
- menyaring setup lemah
- membedakan kualitas antar setup
- membuat indikator lebih jujur
- mengurangi sinyal yang terlalu biner

---

## Ringkasan
Scoring system bukan pengganti logika market.
Scoring system adalah alat bantu untuk menyusun kualitas setup secara lebih terstruktur.
