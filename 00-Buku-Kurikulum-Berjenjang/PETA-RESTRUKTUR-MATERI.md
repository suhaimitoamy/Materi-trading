# Peta Restruktur Materi Lama ke Kurikulum Baru

Dokumen ini menjelaskan bagaimana materi lama dari folder 00–10 dipindahkan secara konseptual ke struktur buku 4 level.

Tujuannya agar materi tidak hilang, tetapi dibaca dalam urutan yang lebih ramah untuk pemula.

---

# Prinsip Pemindahan

Materi tidak disusun ulang berdasarkan folder asal, tetapi berdasarkan tingkat kesiapan murid.

Urutan baru mengikuti pola:

1. **Mengapa** konsep ini penting?
2. **Apa** konsepnya?
3. **Kapan** konsep digunakan?
4. **Bagaimana** menerapkannya?

Konsep yang terlalu advanced ditunda sampai murid punya fondasi teknikal, risk, dan jurnal yang cukup.

# Catatan untuk Pengembangan Selanjutnya

Setelah struktur ini diterapkan, tahap berikutnya adalah mengubah setiap README level menjadi bab-bab penuh.

Prioritas pengembangan:

1. Tulis Bab 1–7 untuk Level 1 secara lengkap.
2. Tambahkan latihan chart sederhana di setiap bab.
3. Tambahkan checklist kelulusan per bab.
4. Tambahkan contoh screenshot chart.
5. Baru lanjut menulis Level 2, 3, dan 4 secara bertahap.

Dengan cara ini, repo akan berkembang menjadi media ajar yang benar-benar bisa dipakai dari beginner sampai advanced.
